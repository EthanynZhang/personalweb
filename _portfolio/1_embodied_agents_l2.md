This PhD research project investigates how Generative AI and Virtual Reality can support spoken second-language learning through embodied conversational agents. The project focuses on pedagogical fidelity: whether agent personalities, teaching strategies, and interaction behaviors meaningfully align with language-learning goals.

The work combines game-engine development, real-time rendering, conversational AI, and empirical evaluation. Current design questions include how learners respond to different agent behaviors, how embodied interaction shapes engagement, and how AI-driven feedback can support oral practice without reducing learning to generic chatbot dialogue.
